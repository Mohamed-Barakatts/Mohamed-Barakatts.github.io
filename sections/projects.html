Ping Pong Game

## Features - Two-player game - Simple scoring system - Realistic paddle and ball movement - Center dashed line

## Controls

  • Player 1: Use `W` to move up and `S` to move down
  • Player 2: Use the `Up` and `Down` arrow keys

    Project #10

    Phone Number Tracking

    A Python script that extracts and displays detailed information about phone numbers using the `phonenumbers` library. This tool allows users to input a phone number with its country code and provides information such as country code, national number, region, country, carrier, and timezone. Features: - Parses and validates phone numbers with the `phonenumbers` library. - Retrieves information such as country code, national number, region, country, carrier, and timezone. - Handles invalid phone numbers gracefully. Usage: 1. Run the script in a Python environment. 2. Enter a phone number with its country code when prompted. 3. View detailed information about the phone number in the terminal. Dependencies: - `phonenumbers` library (install via `pip install phonenumbers`) This script is useful for validating and retrieving basic information about phone numbers, suitable for use in various applications and services that require phone number data.
